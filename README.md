# 🎨 Banner Maker

A simple web-based **banner generator** built with **React**.  
Easily create custom banners, preview them live, and download them instantly.

---

## 🚀 Features
- 🖌️ Live text and color editing
- 🎭 Font and size customization
- 🖼️ Background editor
- ⚡ Real-time preview
- 💾 One-click banner download
- 🌐 SEO-ready with Open Graph & Meta tags

---

## 📦 Prerequisites
Make sure you have these installed on your system:
- **Node.js** ≥ 18.x (recommended: v22 LTS)
- **npm** (comes with Node.js) or **Yarn**
